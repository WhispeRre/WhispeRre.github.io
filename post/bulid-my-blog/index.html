如何搭建自己的第一个静态博客
@ Whisper · Jul 25, 2019 · 1 min read · Update at Jul 25, 2019

How to build my first static blog

1.

首先要安装hugo本地文件：


·第一步,从Hugo Release页面下载hugo的压缩包。
·第二步，在你喜欢的地方创建一个Hugo文件夹：D:\Hugo。
·第三步，将下载的压缩包放到刚刚建立的文件夹里面。
·第四步，解压压缩包文件，提取hugo.exe到’D:\Hugo’里。

·第五步，在你的windows环境变量中加入Hugo.exe路径。

eg:具体安装步骤参考官方安装指南

2.

在D盘中重新创建一个hugodata文件夹,进入cmd，在当前文件夹路径下输入

hugo new site . //生成网站
之后会在文件夹中生成一些空文件夹，此时还没有将主题添加到网页中

·在Hugo Themes中选择主题，可以在主题详情界面对主题进行预览 点击下载，点击Clone or downloads,复制主题链接 ·将主题clone到本地hugodata的themes文件夹中，进入cmd，在当前文件夹路径下输入 
git clone + 主题链接
·将hugodata文件夹中的exampleSite的内容复制到hugodata路径下

·将hugodata文件夹中的themes\hugo-theme-dream的内容复制到hugodata路径下

这样本地的静态网站就算基本搭建完成了

可以在hugodata目录下打开cmd，输入 
hugo server
在浏览器中可以访问localhost:1313，来本地预览创建的网站，这个功能可以用来修改网站内容时在本地进行预览，减少上传次数的时间消耗

但是此时还只是在本地保存网页，并没有同步到云端，接下来就是利用GitHub库来保存

3.

(1)到官网下载GitHub的安装软件


注意进入这个网页后等待几秒钟下载会自动进行，下载完毕后直接双击运行，按照说明安装即可。

(2)安装完成后需要进行的操作：


·首先将Git的安装路径添加到系统的环境变量中：

右击计算机——属性——高级系统设置———环境变量——找到系统变量下的Path，点击新建，将刚刚安装的Git的目录加进去即可：

·在本地cmd中输入命令： 

git --version

如果出现类似下面情况说明安装成功！ 
git version 2.17.0.windows.1

(3)Git初始配置


安装成功后在桌面点击右键——git bash,即可启动git终端

前提要有一个GitHub账户，没有的话可以直接去GitHub官网申请

进入终端后输入以下代码 

git config --global user.name   #这里是官网注册的用户名
git config --global user.email  #这里是官网注册的邮箱
这样就能在本地终端绑定自己的GitHub账户

(4)在GitHub上创建库


进入自己的github主页，点击右上角的+之后选择第一项New repository

填写内容完成库创建(注意：这个库中的内容全部用来存放静态博客的相关内容)

(本人创建了一个名为WhispeRre.github.io的库，这个库名就是在本地存放需要上传到GitHub库的文件夹名，同时也是以后静态博客的网址)

(5)创建本地库


创建一个与GitHub库名相同的文件夹

进入GitHub刚刚创建的库中，点击clone or download,将链接复制下来

在库同名文件夹中右键打开git bash,在命令行输入 

git clone + 库链接
这样就可以把云端Github库的内容复制到本地

(6)将静态博客内容保存到本地库


在之前创建的hugodata文件夹中shift+右键打开命令行，输入 

hugo
这样会同目录下生成一个新的public文件夹，将文件夹内容全选并复制

之后进入库同名文件夹中

注意！！！

观察文件夹中是否有名为.git的文件夹，如果有，则选中除.git之外的文件夹，将刚刚复制的文件夹进行替换

如果没有.git文件夹，查看是否被隐藏，如果被隐藏，则全选库同名文件夹中的全部文件，替换即可

这样之后就将静态博客内容保存到本地库中

(7)将本地库中的文件上传到云端GitHub库中


在本地库文件夹中shift+右键打开命令行，输入 

git add .
git commit -m "尝试上传"
git push
就可以将本地库文件上传到GitHub云端库上

在GitHub库中可以查看自己的上传记录

如果显示了对应的上传记录，则说明上传成功

这时就可以在浏览器中输入自己的域名(比如本人的域名https://whisperre.github.io/) 来查看自己创建的博客

这些步骤过后静态博客的搭建基本上就大功告成了！！！

More.
关于博客界面和内容的修改

·博客界面修改与主题有关，可以在Hugo Themes中找到自己下载的对应主题，点击Homepage 即可查看对应主题的帮助文档，其中包含了一些主题修改的解释与帮助

·博客文章内容在本地一般存放在hugodata\content\post文件夹中，使用markdown语言编辑

Ps.
每次在本地对网站进行修改之后，在hugodata对应的文件夹中打开命令行，输入 

hugo
即可将本地文件中的修改集成在一个新的public文件中
将public文件中的所有内容复制到库同名文件夹中
在库同名文件夹中打开命令行，输入 
git add .
git commit -m "尝试上传"
git push
就完成了一次网站内容的上传
