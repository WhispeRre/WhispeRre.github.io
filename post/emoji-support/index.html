搭建我自己的第一个静态博客
@ Whisper · Jul 25, 2019 · 1 min read · Update at Jul 25, 2019

How to build my first static blog.

1.
首先要安装hugo本地文件：
·第一步,从Hugo Release页面下载hugo的压缩包。
·第二步，在你喜欢的地方创建一个Hugo文件夹：D:\Hugo。
·第三步，将下载的压缩包放到刚刚建立的文件夹里面。
·第四步，解压压缩包文件，提取hugo.exe到’D:\Hugo’里。
·第五步，在你的windows环境变量中加入Hugo.exe路径。
(官方安装指南)

2.
在D盘中重新创建一个hugodata文件夹,进入cmd，在当前文件夹路径下输入 

hugo new site . //生成网站
之后会在文件夹中生成一些空文件夹 在Hugo Themes中选择主题，可以在主题详情界面对主题进行预览 点击下载，进入github界面点击Clone or downloads,复制主题链接 将主题clone到本地的themes文件夹中，进入cmd，在当前文件夹路径下输入 
git clone + 主题链接
·将hugodata文件夹中的exampleSite的内容复制到hugodata路径下
·将hugodata文件夹中的themes\hugo-theme-dream的内容复制到hugodata路径下 这样网站就算基本搭建完成了

可以在hugodata目录下打开cmd，输入 
hugo server
在浏览器中可以访问localhost:1313，来本地预览创建的网站，这个功能可以用来修改网站内容时在本地进行预览，减少上传次数的时间消耗
但是此时还只是在本地保存网页，并没有同步到云端
