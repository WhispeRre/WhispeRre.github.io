如何在虚拟机中安装ubuntu系统
@ Whisper · Jul 26, 2019 · 1 min read · Update at Jul 26, 2019

How to install ubuntu system in virtual machine

1.

准备虚拟机环境


·搜索vmware，进入官网，点击Downloads，并找到Workstation Pro,选择自己想要的版本并下载
·下载完成后按照指示进行安装
·过程中需要输入激活码直接网上搜索即可
·安装完成后虚拟机环境搭建完毕

2.

下载ubuntu镜像


·搜索ubuntu，进入官网，点击Download,选择自己想要的版本并下载

·下载完成后将镜像文件放在vmware创建的虚拟机文件夹中

3.

安装虚拟机


·打开vmware，点击创建新的虚拟机

·选择刚刚安装好的ubuntu光盘镜像


·然后下一步，根据对应提示进行操作，输入自己的用户名和密码，就是以后登录虚拟机系统需要用到的用户名和密码

注意： ·自定义硬件，根据电脑的配置对虚拟机系统硬件进行分配，硬盘空间建议分配额为大于等于20G

·当虚拟机配置完成后，接下来将会开始部署系统

·部署完成后输入对应的用户名和密码即可进入系统

·可以根据网上对应的教程安装相应的中文输入法和进行系统汉化
